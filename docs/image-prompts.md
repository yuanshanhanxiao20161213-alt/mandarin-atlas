# Image Generation Prompts

Generated visual assets live in `public/images/ai/`. They were produced as premium editorial photography for an international Mandarin learning comparison website.

## Shared Visual Direction

- Luxury editorial photography
- Warm ivory, ink black, muted jade, restrained cinnabar accents
- No real logos
- No readable fake app or company names
- No misleading real product screenshots
- Clean negative space for web layout

## Core Prompt Pattern

```text
Create a premium editorial image for an international Mandarin learning comparison website.
Scene: refined Mandarin study, tutoring or resource-comparison context.
Style: luxury editorial photography, calm global education brand, high-end magazine composition.
Lighting: soft daylight, natural shadows, warm highlights.
Color palette: warm ivory, ink black, muted jade, restrained cinnabar red accents.
Constraints: no real brand logos, no readable fake UI brand names, no watermark, no distorted hands, no random text except simple Chinese strokes or tone marks.
```

## People-Focused Additions

- One-on-one Mandarin tutoring with teacher and international adult learner
- Small international classroom with teacher explaining tone marks and character structure
- Modern learning environment with laptops, phones and paper study materials
